
# Wiedza i Życie - Scraper
___________________________

### Prerequisites

* pip
* python3
* virtualenv

___________________________

### Installation

in bash run following command

```bash
make install
```
___________________________

### Building Command

in order to build the command for scraper run:

```bash
python setup.py develop
```
